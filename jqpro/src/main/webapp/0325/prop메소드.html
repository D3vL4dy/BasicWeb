prop() : 속성의 상태값을 얻거나 설정
		상태 : checked, selected, disabled, readonly, multiple
		
		get실행시 리턴값은 true / false
		$(this).prop('checked')
		
		set실행시 true / false로 설정
		$(this).prop('checked', true)
